API Documentation

全局说明 ¶

除下述的公开API，其它私有API均需要在请求头中携带token字段，格式如下

Authenticate: Bearer osAEL0cwqLBL0cwqLBL0cwqLB

其中，不同用户的token的secret秘钥也不同，用户主要分为三大类 user/teacher/admin

URL以 /api 开头的为普通用户请求

URL以 /teacher/api 开头的为教师用户请求

URL以 /admin/api 开头的为管理员用户请求

后端应对除公开API外的其他接口进行正则匹配以完成不同的token验证

HTTP状态码 ¶

  • 200 成功

  • 401 认证失败(非注册用户)

  • 403 无权限

  • 500 服务端错误

公开API列表 ¶

  • /api/wx_login [POST]

  • /admin/api/campuses [GET]

  • /admin/api/academies [GET]

  • /admin/api/professions [GET]

  • /admin/api/classes [GET]

  • /admin/api/login [POST]

约定 ¶

  • POST请求（新建/更新）返回数据库一整条数据

  • GET请求则按需返回数据
